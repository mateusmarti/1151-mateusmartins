O que é HTML?

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

"main": O elemento em seu documento ou aplicação. Entende-se como conteúdo principal relacionado ou diretamente com o tópico central da página com a funcionalidade central da aplicação.


"nav": O Elemento HTML de Navegação representa uma seção de uma página ou páginas que aponta para outras áreas da página, ou seja, uma seção com links de navegação.


"a href": Serve para criar um texto em forma de link, e redirecioná-lo para outra página.
